题目描述

GG 和 MM 从小就爱一起玩游戏。在游戏开始，有两堆石子。MM 可以选择其中一堆，设其中有 $x$ 个石子，然后她会选择一个正整数 $k$ 并且在另一堆石子 (设有 $y$ 个) 中取走 $k \cdot x$ 个，当然，要满足 $y \geq k \cdot x$。

接下来轮到 GG，按照同样的规则去石子。当一个人无法再取走任何石子时，她就输了，并且这个游戏到此结束。

几年以后，GG 和 MM 发现这个游戏太简单了，于是她们决定同时玩 $N$ 个游戏。MM 先手，类似地，每一轮中，操作者需要将每一个未结束的游戏都操作一遍。具体操作规则与前面相同。

当某个人无法再取走任何游戏中的任何石子 (亦即输掉了最后一个游戏) 时，她就输了。我们可以假设 GG 和 MM 足够聪明，请判断谁会赢。

输入格式

输入包含多组数据 (不超过 $100$ 组)。

每组数据的第一行包含一个正整数 $N$ ($N \leq 1000$)，表示同时进行的游戏个数。

接下来的 $N$ 行，每行包含两个非负整数 $p, q$ ($p, q \leq 1000$)，表示每个游戏中两堆石子的个数。

输入终止于 EOF。

输出格式

对于每组输入数据，输出一行，为胜利者的名字 (GG 或 MM)。

题解

代码

#include <bits/stdc++.h>
#define N 50

int n, a[N];

inline void up(int &x, const int y) {x < y ? x = y : 0;}

void work() {
	int i, j, la, cnt, max = 0;
	for (; n; --n) {
		scanf("%d%d", a, a + 1); if (*a < a[1]) std::swap(*a, a[1]);
		for (cnt = 1; a[cnt]; ++cnt, a[cnt] = a[cnt - 2] % a[cnt - 1]);
		j = cnt - 1; la = -1;
		for (i = 0; i < cnt - 1; ++i)
			if (a[i] >= 2 * a[i + 1])
				j += ~la && (i ^ la) & 1, la = i;
		up(max, j);
	}
	puts(max & 1 ? "MM" : "GG");
}

int main() {
	int T;
	while (scanf("%d", &n) && n) work();
	return 0;
}
