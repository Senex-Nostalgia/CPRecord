Euler 变换 —— 无标号计数中的 exp

组合计数，是一项悠久且古老的分支。所用到的知识非常灵活，如有 DP、组合数学的结论、数两次、图论等众多方法。其中，生成函数 (Generating function) 是解决这类问题的一大利器。

在生成函数中，比较著名的是普通生成函数 (Ordinary g.f.) 和指数生成函数 (Exponential g.f.)，它们通常用来解决无标号计数和有标号计数的问题。

在有标号计数中，众所周知有一种操作叫做求指数 (exp)，它的一个易于理解的形式如下：

设对于一个大小为 $i$ 的组，其内部有 $a_i$ 不同方案。设 $\left\{ a_n \right\}$ 的指数生成函数为 $A \left( x \right)$，则 $B \left( x \right) = 1 + A \left( x \right)+ \dfrac {A^2 \left( x \right)} {2 !} + \dfrac {A^3 \left( x \right)} {3 !} + \cdots + \dfrac {A^k \left( x \right)} {k !} + \cdots = \mathrm e^{A \left( x \right)} = \exp A \left( x \right)$ 表示将一个有序序列分成若干个无序组 (盒子与盒子之间无区别) 的方案数。

具体地，$n ! \left[ x^n \right] \exp A \left( x \right)$ 表示将长度为 $n$ 的序列 (如 $\left\{ 1, 2, \cdots, n \right\}$) 分成若干个无序组的方案数。

不过，这里我们需要一个假设：$\left[ x^0 \right] A \left( x \right) = 0$ ($A \left( x \right)$ 的常数项为 $0$)。一方面，为了保证 $\exp$ 出来的结果是有理数；另一方面，在组合意义上，去除 "空组" 对计数的影响。

稍微变换一下，如果要将长度为 $n$ 的序列分成若干个有序组 (盒子与盒子之间有区别)，那么方案数就是 $1 + A \left( x \right) + A^2 \left( x \right) + \cdots + A^k \left( x \right) + \cdots = \dfrac 1 {1 - A \left( x \right)}$。

这种计数往往会在竞赛图的方面出现，比如有标号强连通竞赛图的计数。其中同样需要保证 $\left[ x^0 \right] A \left( x \right) = 0$。

在无标号计数中，就不再需要考虑顺序的问题，只是一些基本元件之间的互相 "组合"。

如果这些元件之间讲究顺序 (盒子与盒子之间有区别)，那么 $n$ 个元件的组合方案数自然就是 $A^n \left( x \right)$。从而总的方案数平凡地就应该是 $1 + A \left( x \right) + A^2 \left( x \right) + \cdots + A^k \left( x \right) + \cdots = \dfrac 1 {1 - A \left( x \right)}$。

那如果，这些元件之间是无序 (盒子与盒子之间无区别) 的呢？这就是这篇文章所讨论的重点，下面就开始正题。

前置技能：Pólya 计数相关、生成函数基础、多项式全家桶。

对于一个形式幂级数 $\displaystyle A \left( x \right) = a_1 x + a_2 x^2 + a_3 x^3 + \cdots = \sum_{i \geq 1} a_i x^i$，定义它的欧拉变换 (Euler transform) 是这样一个形式幂级数 $B \left( x \right)$，满足

$$ B \left( x \right) = \prod_{i \geq 1} \frac 1 {\left( 1 - x^i \right)^{a_i}} \tag 1 \label 1 $$

下面说明，这样一个表达式就是我们所要求的 "无标号计数中，元件之间互相形成无序组" 的方案数。
